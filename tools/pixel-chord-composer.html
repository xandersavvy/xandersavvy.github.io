How it works

Pixel regions → chords: Image regions map to complete chords using music theory progressions.

Colour harmony → chord extensions: RGB values determine 7ths, 9ths, and other extensions.

Brightness → inversions: Pixel brightness controls chord voicing and inversions.

Position → progressions: Image flow generates musically coherent chord sequences that follow harmonic rules.
